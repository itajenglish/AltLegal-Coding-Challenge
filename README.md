# Twitt-Watch-App

React App For Watching Twitter Feeds For Your Favorite Hashtags.
------------
### Setup Instructions
`Run $ npm install inside root folder`

` Start Api Server` - https://github.com/itajenglish/Twitt-Watch-Backend

`Run $ npm start`

`Enjoy :D`

### Technologies used:
- React
- react-spinjs
- Axios
